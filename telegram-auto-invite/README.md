# 🤖 Telegram Auto Invite Bot

Автоматична система для парсингу учасників з однієї Telegram групи та додавання їх до вашого каналу.

## 🚀 Швидкий старт

### Крок 1: Встановлення Python

1. Завантаж Python 3.8+ з [python.org](https://www.python.org/downloads/)
2. При встановленні **обов'язково відміт** "Add Python to PATH"

### Крок 2: Встановлення залежностей

Відкрий PowerShell у цій папці та виконай:

```powershell
pip install -r requirements.txt
```

### Крок 3: Отримання API credentials

1. Відкрий https://my.telegram.org/apps
2. Увійди зі своїм номером телефону
3. Створи новий додаток (будь-яка назва)
4. Скопіюй `api_id` та `api_hash`

### Крок 4: Налаштування

Відкрий файл `config.py` та заповни:

```python
API_ID = '12345678'  # Твій api_id
API_HASH = 'abcdef1234567890'  # Твій api_hash
PHONE = '+380123456789'  # Твій номер телефону
```

Інші налаштування (група-донор, твій канал) вже заповнені.

### Крок 5: Парсинг учасників

```powershell
python parser.py
```

Скрипт:
- Підключиться до Telegram (перший раз попросить код з SMS)
- Витягне всіх учасників з групи-донора
- Збереже їх у `parsed_members.json`

### Крок 6: Додавання учасників

```powershell
python inviter.py
```

Скрипт:
- Почне додавати користувачів до твого каналу
- Робить затримки 30-60 секунд між додаваннями
- Логує результати у `invite_log.txt`
- За один запуск додає максимум 50 осіб

## ⚙️ Налаштування

У файлі `config.py`:

### Швидкість додавання
```python
ADD_DELAY_MIN = 30  # Мінімальна затримка (секунди)
ADD_DELAY_MAX = 60  # Максимальна затримка (секунди)
MAX_ADDS_PER_SESSION = 50  # Максимум додавань за раз
```

⚠️ **Не роби затримки менше 30 секунд!** Це може призвести до бану.

### Кількість парсингу
```python
PARSE_LIMIT = 1000  # Максимум учасників (None = всі)
```

## 📊 Результати

### `parsed_members.json`
Список всіх спарсених користувачів:
```json
[
  {
    "id": 123456789,
    "username": "example_user",
    "first_name": "Іван",
    "last_name": "Петров"
  }
]
```

### `invite_log.txt`
Лог додавань:
```
[2026-02-11 11:30:15] @user1 - ✅ Успішно додано
[2026-02-11 11:31:20] @user2 - ❌ Приватність (користувач заборонив додавання)
[2026-02-11 11:32:45] @user3 - ✅ Успішно додано
```

## ⚠️ Важливо

### Обмеження Telegram

- **Щоденний ліміт**: ~50-200 додавань (залежить від віку акаунта)
- **FloodWait**: Telegram може попросити почекати
- **PeerFlood**: При надто активному використанні - бан на 24+ години

### Рекомендації

1. **Не поспішай**: Використовуй затримки 30-60 секунд
2. **Поступово**: Додавай по 50 осіб на день
3. **Зупиняйся**: Якщо отримав `PEER_FLOOD` - почекай день
4. **Новий акаунт**: Перші дні ліміт буде нижчим

### Безпека

- ❌ Не діли свій `api_hash` та `api_id`
- ❌ Не запускай на чужих серверах
- ✅ Зберігай `telegram_session.session` у безпеці (це твій логін)

## 🔧 Troubleshooting

### "Модуль telethon не знайдено"
```powershell
pip install telethon
```

### "API_ID or API_HASH invalid"
Перевір правильність `api_id` та `api_hash` у `config.py`

### "You are banned from this chat"
Твій акаунт забанений у групі-донорі. Спробуй іншу групу.

### "PEER_FLOOD" 
Твій акаунт тимчасово обмежений. Почекай 24-48 годин.

## 📝 Workflow

Типовий робочий процес:

1. **Ранок**: `python parser.py` (парсинг)
2. **Протягом дня**: `python inviter.py` (по 50 осіб кожні 2-3 години)
3. **Наступний день**: Повтор
