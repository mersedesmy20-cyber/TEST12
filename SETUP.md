# Інструкція по запуску проекту

## Крок 1: Встановлення залежностей

Відкрийте термінал (PowerShell або Command Prompt) в папці проекту та виконайте:

```bash
npm install
```

Якщо npm не встановлений, встановіть Node.js з https://nodejs.org/

## Крок 2: Запуск dev сервера

Після встановлення залежностей запустіть:

```bash
npm run dev
```

## Крок 3: Відкрити браузер

Після запуску ви побачите повідомлення:
```
- ready started server on 0.0.0.0:3000, url: http://localhost:3000
```

Відкрийте браузер та перейдіть на: **http://localhost:3000**

## Можливі проблеми

### Помилка: "npm не розпізнається"
- Встановіть Node.js з https://nodejs.org/
- Перезапустіть термінал після встановлення

### Помилка: "Port 3000 is already in use"
- Закрийте інший процес, який використовує порт 3000
- Або запустіть на іншому порті: `npm run dev -- -p 3001`

### Помилка при встановленні залежностей
- Перевірте, що у вас встановлений Node.js версії 18+
- Спробуйте видалити `package-lock.json` та `node_modules` (якщо є) і встановити знову

## Альтернативні команди

Якщо використовуєте yarn:
```bash
yarn install
yarn dev
```

Якщо використовуєте pnpm:
```bash
pnpm install
pnpm dev
```
